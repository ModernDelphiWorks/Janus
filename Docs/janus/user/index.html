Manual do Usuario Janus

Manual do usuario do Janus para Delphi, com foco em instalacao, primeiros fluxos de uso e todas as features do framework.

Estado atual do manual: alinhado com a release v2.19.4, incluindo lazy loading transparente em ObjectSet, DataSet e REST. As correcoes da R17.2 e R17.3 permanecem internas (hardening, estabilidade e governanca documental), sem mudancas no contrato publico de uso do framework.

Para quem este manual foi feito​

  • Desenvolvedores Delphi que precisam persistir objetos em banco relacional.
  • Equipes que querem reduzir SQL manual em operacoes de CRUD.
  • Operadores tecnicos que mantem aplicacoes VCL/FMX com DataSet e rotinas de integracao.

Jornada recomendada​

  • Introducao
  • Primeiros passos

Guias por feature​

Persistência e dados​

  • Primeiro CRUD com DataSet — fluxo visual VCL/FMX
  • Operacao Master-Detail — TManagerDataSet
  • ObjectSet (sem DataSet visual) — para servicos e APIs
  • Consultas personalizadas — filtros e paginacao via FluentSQL (TCQ)

Tipos especiais​

  • Campos opcionais com Nullable — Nullable<T> para campos NULL
  • Lazy Loading — carregamento adiado com proxy transparente e compatibilidade com LoadLazy

Interface e binding​

  • LiveBindings VCL/FMX — binding automatico via atributos
  • Monitor SQL — diagnostico de comandos em tempo real

Features avancadas​

  • Eventos Before e After — interceptar operacoes DML
  • CodeGen — gerar entidades a partir do banco
  • Serializacao JSON — converter entidades para/de JSON
  • Driver RESTful — persistencia via HTTP
